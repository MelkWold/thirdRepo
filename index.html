A hyphenated introduction of HTML

Hypertext Markup Language (HTML) is a powerful and versatile web programming language that allows users define the structure and style of websites as well as the way website elements are displayed on browsers. In short, HTML is the basic building block of all webpages.

In addition, HTML can be integrated with other powerful programming languages like CSS and JavaScript to produce beautiful and interactive websites, respectively. In addition, frameworks like React, Angular and Vue are often used with HTML while developing webpages.

Some basic vocabulary that you need to know before continuing include:

  • Uniform Resource Locator (URL): refers to the address of a website on a server. E.g. https://www.google.com
  • Server: a repository that stores all the data associated with websites, receives and processes requests sent from clients and sends the appropriate responses back to the client.
  • Client: any one that sends a request to a server, e.g. using a web browser or via API
  • Document Object Model (DOM): the objects or the elements in an HTML document. E.g. paragraphs, links, images, tables, etc
  • Element: any item in the the HTML document including the head, the body, the paragraphs, the images, the tables etc

The basic HTML Template

Any HTML file is based on the following basic template also known as boilerplate code:

    Description of the components of the HTML template

  • "!DOCTYPE html": denotes the version of HTML used and informs the HTML validator to follow the HTML standards when checking the code.
  • "html": instructs the web browser that the document is an HTML document and indicates the start of the HTML document (a.k.a. the root of the document.
  • "head": contains the necessary metadata including title, character encoding, language and the like. Note that the metadata of a web document are not displayed on the web browser.
  • "body": contains all the data and content that is going to be displayed on the web site

HTML Tags

HTML tags are the fundamental building blocks of an HTML document and are used to format the contents of a website and determine its structure and style. Most HTML tags begin and end with angle brackets ('<>') with the respective element contained in between.

Note: In addition to the tags displayed below, there are tags that do not require the closing angle bracket known as "self-closing tags" or "void tags" . Some examples of void tags include: img, br, source, input, embed
