A hyphenated introduction of HTML

Hypertext Markup Language (HTML) is a powerful and versatile web programming language that allows users control the structure and style of websites as well as the way website elements are displayed.

In addition, HTML can be integrated with other powerful programming languages like CSS and JavaScript to produce interactive and beautiful websites. What makes HTML particularly remarkable is the relative simplicity to learn. This document intended to be a beginner's guide and will remain to be a work in progress.

The basic HTML Template

Any HTML file is based on the following basic template:

    Description of the components of the HTML template

  • "!DOCTYPE html": denotes the version of HTML used and informs the HTML validator to follow the HTML standards when checking the code.
  • "html": instructs the web browser that the document is an HTML document and indicates the start of the HTML document (a.k.a. the root of the document.
  • "head": contains the necessary metadata including title, character encoding, language and the like. Note that the metadata of a web document are not displayed on the web browser.
  • "body": contains all the data and content that is going to be displayed on the web site

HTML Tags

HTML tags are fundamental building blocks that are used to prepare HTML documents. Tags are used to format the contents of a website and determine its structure and style. Most HTML tags begin and end with angle brackets ('<>') with the respective element contained in between.
